# Digital David VideoAsk Funnel Script

**Purpose:** Multi-purpose lead qualification, product discovery, and booking funnel
**Tone:** Bold, confident, innovative - matches Digital David brand
**Speaker:** Stefan Rühle (or team member)

---

## Funnel Overview

```
START (Welcome)
    │
    ├─► "I want to learn about your products" ──► PRODUCT DISCOVERY BRANCH
    │
    ├─► "I have a project in mind" ──► PROJECT QUALIFICATION BRANCH
    │
    └─► "I want to talk to someone" ──► DIRECT BOOKING BRANCH
```

## VideoAsk Settings

### Recommended Configuration

| Setting | Value |
|---------|-------|
| Widget Type | Popup or Embed |
| Trigger | On button click (not auto-open) |
| Branding | Custom colors: #0A0F1C (deep-space), #6366F1 (electric-blue) |
| Thumbnail Style | Circle with play button |
| Response Types | Multiple choice, Video, Text, Calendar |
| Language Detection | Auto (based on browser) or manual toggle |

### Integrations

| Platform | Purpose |
|----------|---------|
| Calendly / Cal.com | Booking slots |
| HubSpot / Pipedrive | CRM sync (contacts + tags) |
| Slack | Notifications for new responses |
| Zapier | Custom workflows |

### Analytics to Track

- Completion rate per node
- Drop-off points
- Most selected paths
- Conversion: VideoAsk start → Booking
- Language split (DE vs EN)
